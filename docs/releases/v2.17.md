## #39 — self-hosted the display fonts (no third-party CDN)
`index.html` loaded Cinzel + Cormorant Garamond from `fonts.googleapis.com` / `fonts.gstatic.com`
on every page load — leaking the visitor's IP to Google, render-blocking on a third-party host, and
with no offline fallback. Both are now **self-hosted**: the latin-subset woff2 (the only glyphs the
English game uses) live in `src/fonts/`, bundled + content-hashed by Vite into `/assets/` (so they
inherit the `immutable`, year-long cache) and referenced by `@font-face` in `src/index.css`. Both
are **variable fonts**, so one file per family covers every weight the game requests — exactly what
Google served (verified: Google returned one latin file for Cinzel 400–900 and one for Cormorant
400/500). The Google `<link>` preconnects + stylesheet are gone; the page now makes **zero
third-party requests**.

## #40 — favicon, PWA manifest, apple-touch-icon, social meta
- A code-drawn `public/favicon.svg` — a grace-gold halo + falling blade on the `#0b0907` ground (the
  game's own palette; still zero hand-drawn art).

  description, and preview image.
