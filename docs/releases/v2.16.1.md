## #37 ‚Äî removed the unreachable shadcn tree + 43 prod deps
The rendered surface is `main.tsx ‚Üí App.tsx ‚Üí pages/Home.tsx`, which imports only React,
`react-router`, and the canvas engine. `src/components/ui/` (the full 50-component shadcn/ui
library), `src/hooks/use-mobile.ts`, `src/lib/utils.ts`, `src/App.css`, and `components.json` were
all unreachable. Confirmed by grep (no `@/components/ui`, `@/hooks/use-mobile`, or `@/lib/utils`
import outside the dead tree) and **proven by the build** ‚Äî `tsc` fails on any live import of a
deleted file.

Deleted:
- `src/components/` (50 files), `src/hooks/use-mobile.ts`, `src/lib/utils.ts`, `src/App.css`,
  `components.json`.
- **43 production dependencies** ‚Äî every `@radix-ui/*` primitive plus `cmdk`, `recharts`,
  `embla-carousel-react`, `react-day-picker`, `date-fns`, `vaul`, `react-hook-form`,
  `@hookform/resolvers`, `zod`, `input-otp`, `react-resizable-panels`, `next-themes`, `sonner`,
  `lucide-react`, `class-variance-authority`, `clsx`, `tailwind-merge`. **Kept:** `react`,
  `react-dom`, `react-router`.

`src/` is now the six files that are the actual game (`main`, `App`, `pages/Home`, `game/engine`,
`game/audio`, `index.css`). The production **CSS bundle drops 80 KB ‚Üí 10.6 KB** (all unused
shadcn/tailwind utility CSS); the JS bundle is unchanged. `index.css` (with the load-bearing
`--sa-b`/`--sa-r` safe-area vars) and `App.tsx` are **kept** ‚Äî both are live. `tailwindcss-animate`
(a devDep required by `tailwind.config.js`) is kept.
